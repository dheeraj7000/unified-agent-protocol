# UAP Development Roadmap

This document outlines the planned trajectory of the Unified Agent Protocol (UAP) implementation.

## Phase 1: v0.1 (Current)
- Fully functional reference implementation.
- Conformance runner and 14 base test vectors.
- Strict scope checking and local approval gating lifecycle.

## Phase 2: v0.2
- **JWT-Backed Delegation**: Implement decentralized authorization and validation of actor token scopes.
- **Real MCP Server Bridge**: Expand adapters into dual-direction bridges mapping MCP servers directly.
- **PyPI Distribution**: Packages published and installable via `pip install uap-reference`.

## Phase 3: v1.0 (Stable release)
- **Stable Spec**: Frozen schemas, standard REST and gRPC bindings.
- **Full Conformance Suite**: Automated testing matrix across multiple runtime engines.
- **Multi-Runtime Interoperability**: Live demonstrations bridging distinct protocol agents in a single workflow.
